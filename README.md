# Candy Crush
## Descripción
Consiste en simular el conocido juego de ***Candy Crush***.
## Versiones
### V1.0
Esta versión emula el Candy Crush por consola, tiene un nivel, 5 dulces, sistema de puntuación, vidas, movimientos y una interfaz agradable.
