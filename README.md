# Candy Crush

## Descripción
Consiste en simular el conocido juego de ***Candy Crush***.

## Versiones
### V1.0.1 Candy Crush por consola 🍬 🍬 🍬 
Esta versión emula el Candy Crush por consola.  
Esta versión añade:  
🍬 Un nivel.  
🍬 5 tipos de dulces.  
🍬 Sistema de puntuación.  
🍬 Sistema de vidas.  
🍬 Interfaz por consola.
